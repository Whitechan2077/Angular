Component là gì ?:
  • Là kết hợp giữa template với properties & method + metadata
  • 1 Component là một class chứa View và các thao tác với view giống như JFrame của Swing
  • Trong Component có
    • selector: tên thẻ để đẩy html vào ?(Hỗ trợ SPA)
    • template: tên file view cần đổ vào selector
    • styleUrl: file css riêng cho Component
  • Trong class của component sẽ code nhưng hàm sử lý logic ở trên view
  • Gắn kết dữ liệu cách 1 bằng: {{x}}
  • Gắn kết dữ liệu cách 2 bằng: {{x1}} Nhớ import trong Module angular core tên module là model
  • Sử lý sự kiện: {{evt}}

  • DIRECTIVE - chỉ thị
    • *ng-if: thẻ chỉ thị điều kiệu if
    • *ng-for: vòng lặp fore
